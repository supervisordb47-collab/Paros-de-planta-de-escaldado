Sincronización con GitHub

La nube usa la API de GitHub Contents. El token se guarda solo en esta sesión, así que conviene usar uno de alcance mínimo.

Acceso y privilegios
  • ADMIN puede crear, editar, deshabilitar y eliminar usuarios.
  • ADMIN puede editar y eliminar todos los registros.
  • Usuarios normales trabajan sobre sus propios registros.
  • Las notificaciones se guardan por sesión y se exportan con el respaldo.
